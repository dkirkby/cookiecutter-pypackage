============
Installation
============

At the command line::

    $ easy_install {{ cookiecutter.repo_name }}

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv {{ cookiecutter.repo_name }}
    $ pip install {{ cookiecutter.repo_name }}
